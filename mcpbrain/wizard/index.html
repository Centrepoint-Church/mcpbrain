mcpbrain setup

Connect your Google account and link mcpbrain to Claude Desktop. Everything stays on this machine.

1Connect Google

Grant read-only access to Gmail and Drive (and Calendar if available). A browser tab opens for consent.

Not connected

Connected means a credential is stored on this machine. If syncs fail later, run this step again.

2Enrichment (optional)

A Gemini API key builds the knowledge graph behind richer answers. Skip to leave it off; search still works.

3Register with Claude Desktop

Adds mcpbrain to Claude Desktop as a tool. Existing tools are kept.

After registering, quit Claude Desktop completely and reopen it.

4Status

Live state of the background daemon.
