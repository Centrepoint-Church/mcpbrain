mcpbrain setup

Connect your Google account and link mcpbrain to Claude Desktop. Everything stays on this machine.

1Connect Google

Grant read-only access to Gmail and Drive (and Calendar if available). A browser tab opens for consent.

Not connected

Connected means a credential is stored on this machine. If syncs fail later, run this step again.

2About you (optional)

Your name, email, and role drive action attribution and the extraction prompts. The organisations you work across (work, study, church, personal…) become the org tags everything is classified against; email domains are optional and help classify senders. Leave blank to keep the defaults.

3Search model

A small on-device embedding model (~130 MB) powers search. It downloads once and stays on this machine.

Checking…

4Connect Claude Desktop

Do this last. It restarts Claude Desktop so it loads your brain — any open Claude Desktop chat will close and reopen.

Ready when you are

5You're set up

The brain is installed and running on this Mac. The recurring background tasks (enrich, meeting-packs, gardener, reference-gardener) are created back in the Claude Code session that installed mcpbrain — as Local scheduled tasks. There's nothing to attach here: the brain is served through its MCP tools, so just start asking questions in Claude or Cowork.

6Backup & recovery

Automatic. Once you connect Google, mcpbrain either restores your brain if a backup already exists for your account, or turns on encrypted backup if it's a fresh start. Your data never leaves in cleartext — only an admin-escrowed key can restore it. No setting to choose.
